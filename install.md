#  Installation instructions:

## Brief version

If you are already familiar with Javelin:

1. Import Script, then Import layout
2. Build and flash firmware

## Detailed version

### Keeb
At the time of writing, this repo only works for the Jarne keyboard.  Note that Javelin supports many other boards.


### Javelin
Go to the [Javelin firmware builder](https://lim.au/#/software/javelin-steno) and select Jarne variant B.

![alt text](<docs/images/Screenshot 2026-03-15 164813.png>)

### Select 'expert options'.

![alt text](<docs/images/Screenshot 2026-03-15 164833.png>)

### Defaults
Accept the defaults generally.

### Dictionaries
Add your dictionaries, or accept Plover/Lapwing defaults, read the descriptions, and add system files.

### Save. Save. Save.

Don't forget to download both your layout and script files when you're done, and save them somewhere (just in case!).  3-dots at top-right of screen.

![alt text](<docs/images/Screenshot 2026-03-15 170256.png>)
   
### Click 'Create Firmware'.
The firmware is created locally in your browser (yes, incredible, right?) and your browser will download the `.uf2` file to your desktop.  Follow the Jarne instructions to load the firmware onto your keyboard and test.

https://lim.au/#/hardware/jarne-manual
